Thunderbird 入门

Apr 5, 2014 | 工具

    Mozilla Thunderbird是由Mozilla浏览器的邮件功能部件所改造的邮件工具，使用 XUL 程序界面语言所设计，介面设计更简洁、而且免安装.下面是一些在使用Thunderbird邮件客户端时总结的经验,适用于每天收发海量邮件的用户.

消息过滤器

所谓消息过滤器是一种允许用户根据邮件头信息对收到的邮件进行分类管理的程序.点击Thunderbird菜单栏的工具下拉菜单,可以看到消息过滤器,过滤规则默认提供了”主题”,”收件人”,”发件人”,”日期”,”消息体”等,可以按照自己需求设定,最为关键的是还提供了”自定义功能”,所谓自定义功能就是允许用户自己按照邮件头信息去定义.选择一份收到的邮件,点击菜单栏 -> 查看消息源代码,就可以看到消息头信息,如下所示:


    .....
    X-Account-Key: account1
    X-UIDL: 1701758
    X-Mozilla-Status: 0000
    X-Mozilla-Status2: 00000000
    X-Mozilla-Keys:                                                                                 
    To: 
    From: 
    Subject: 演示
    Content-Type: text/html; charset="utf8"
    Return-Path: chaubeau.github.io
    X-MS-Exchange-Organization-AuthSource: chaubeau.github.io
    X-MS-Exchange-Organization-AuthAs: Anonymous
    X-Auto-Response-Suppress: DR, OOF, AutoReply
    MIME-Version: 1.0

    可以看到,这是一组KV存储的数据,其中的Key我们就可以用来自己去定义消息过滤规则.Value用来作为过滤的条件.Thunderbird灵活的过滤规则当然也对应了很多的处理动作,比如”移动消息”,”复制消息”,”转发消息”等.当然Thunderbird的过滤规则也有其不足的地方,比如(A or B) and C,这样的规则,就需要拆分为A and C和B and C这样的条件,增加了额外的工作.

邮件保留策略

当你每天接受上千份邮件的时候,你就要考虑这些邮件如何保存,那些需要永久保存,对于不需要永久保存的邮件,手动去清理么?不,Thunderbird提供了方案,右键收件箱下面的文件夹,选择保留策略,Thunderbird提供了四种策略:

  • 不要删除任何消息.
  • 除最近使用的N条消息外全部删除.
  • 删除超过N天的消息.
  • 总是保留加星表邮件.

邮件模板

邮件模板的功能就是提供一个撰写,回复邮件的模板,比如问候语,签名,邮件格式等.使用该功能需要添加SmartTemplate这个附加组件.下面分别提供新建,回复,转发的模板,供大家参考.
新建邮件模板:

<span style="color:#003399;font-family:Tahoma;">
Hi,%cursor% :<br>
<br>
<br>
Thank you <br>
Chaubeau<br>
<div style="margin:0;padding:0; width:960px;height:1px;background-color:#003399;overflow:hidden;">
电话: &ensp;010-XXXXX<br>
手机: &ensp;+86 XXXXXX<br>
邮编: &ensp;100010<br>
地址: &ensp;北京市XXXXXX<br>
</div>
</span>

回复模板:

<span style="color:#003399;font-family:Tahoma;">
Hi,%from(name)%:
    <br>%cursor%
    <br>
Thank you<br>
Chaubeau<br>
%Y%-%m%-%d% %H%:%M%<br>
<div style="margin:0;padding:0; width:960px;height:1px;background-color:#003399;overflow:hidden;">
</div>
</span>

转发模板:

<span style="color:#003399;font-family:Tahoma;">
<br>
转发 来自%from(name)%的 "%subject%"<br>
<br>
%Y%-%m%-%d% %H%:%M%<br>
Chaubeau
<div style="margin:0;padding:0; width:960px;height:1px;background-color:#003399;overflow:hidden;">
</div>

表格

之所以把Thunderbird的表格拿出来单独讲,是因为对Thunderbird默认提供的表格样式不满意,但是Thunderbird本身没有提供预设表格样式的功能,我们可以通过一个比较另类的方法解决:邮件签名,之所以能通过这个设置表格样式,是因为Thunderbird的邮件签名提供了使用HTML的选项,我们在签名文字里面填写如下内容,这样就定义了我们在Thunderbird中的表格样式.

<style>
table
  {
  border-collapse:collapse;
  }
table, td, th
  {
  border:1px solid green;
  }
th
  {
  background-color:green;
  color:white;
  }
table
  {
  width:70%;
  }
</style>

附件组件

Thunderbird有丰富的附件组件可以使用,可以按照自己的需求去添加,推荐如下几个组件:

  • ReminderFox
  • MinimizeTOTray
  • Lightning
